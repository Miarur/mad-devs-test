# mad-devs-test

### Clone repo 
```bash
https://github.com/Miarur/mad-devs-test.git
```

### Go to folder
```bash
cd mad-devs-test
```

### Install all dependencies
```bash
npm i
```

### Run the project
```bash
npm run dev
```

### Build the project
```bash
npm run build
```

### Run tests 
```bash
cd src/spec

npx jest
```
